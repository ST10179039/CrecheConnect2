## Admin Features

### Navigation Tabs
1. **Home** - Dashboard with statistics
2. **Parents** - Manage parent information
3. **Children** - Manage child profiles
4. **Attendance** - Mark daily attendance
5. **Events** - Create and manage events
6. **News** - Post announcements

### Parent Management
- **Add Parent**: Capture name, contact info, address, emergency contacts
- **Edit Parent**: Update any parent information
- **Delete Parent**: Remove parent (also deletes their children)

### Child Management
- **Add Child**: Capture comprehensive details including:
  - Basic info (name, DOB, gender)
  - Parent selection
  - Medical info (allergies, conditions, medications)
  - Doctor details
  - Medical aid information
  - Emergency contacts
  - Special needs & dietary restrictions
- **Edit Child**: Update any child information
- **Delete Child**: Remove child from system

### Key Information to Capture

#### For Parents (Required: *)
- Personal: First Name*, Last Name*, Email*, ID Number
- Contact: Mobile Phone, Work Phone
- Address: Street Address, City, Postal Code
- Emergency: Primary & Secondary contacts with relationships

#### For Children (Required: *)
- Basic: First Name*, Last Name*, DOB*, Gender, Parent*
- Medical: Blood Type, Allergies, Chronic Conditions, Medications
- Doctor: Name, Phone
- Medical Aid: Provider, Number
- Emergency: Contact (if different from parent)
- Other: Special Needs, Dietary Restrictions

## Quick Actions

### Daily Workflow (Admin)
1. Login as admin
2. Go to Attendance tab
3. Mark present/absent for each child
4. Check for any new announcements to post
5. Review upcoming events

### Adding a New Family
1. Go to Parents tab
2. Click "+ Add Parent"
3. Fill in all parent details
4. Save parent
5. Go to Children tab
6. Click "+ Add Child"
7. Select the parent from dropdown
8. Fill in all child details
9. Save child

### Emergency Situations
1. Open child's profile
2. Check emergency contact information
3. Call parent first
4. If unavailable, call emergency contact
5. Check for allergies/medical conditions
6. Have medical aid information ready

## Data Entry Tips

### Date Format
- Always use: YYYY-MM-DD
- Example: 2019-03-15

### Phone Numbers
- Include country code: +27123456789
- Or use local format: 0123456789

### Allergies
- Be specific: "Peanuts, Tree nuts" not just "Nuts"
- List all known allergies

### Medications
- Include dosage: "Ventolin inhaler - 2 puffs as needed"
- Note frequency and conditions

### Blood Types
- Use standard notation: A+, A-, B+, B-, AB+, AB-, O+, O-

## Troubleshooting

### Can't see data
- Pull down to refresh
- Check internet connection
- Verify you're logged in

### Can't add child
- Make sure parent exists first
- Check all required fields are filled
- Verify date format

### Changes not saving
- Check for error messages
- Verify all required fields
- Try again after a few seconds
