## Important Reminders

### Allergies ‚ö†Ô∏è
- Be VERY specific (e.g., "Peanuts, Tree nuts" not just "Nuts")
- Ask about severity
- Document any previous reactions
- Note if EpiPen or other emergency medication is required

### Medications üíä
- Include exact dosages
- Note frequency (e.g., "twice daily", "as needed")
- Document storage requirements
- Confirm expiry dates

### Emergency Contacts üìû
- Verify all phone numbers work
- Ensure contacts are aware they're listed
- Confirm they can be reached during school hours
- Get alternative contact methods if possible

### Medical Aid üè•
- Verify coverage is active
- Note any pre-authorization requirements
- Document emergency procedures
- Keep copy of medical aid card

### Dietary Restrictions üçΩÔ∏è
- Distinguish between allergies and preferences
- Note severity of restrictions
- Document safe alternatives

## Verification Steps

Before completing registration:

1. **Contact Information**
   - [ ] Call mobile number to verify
   - [ ] Call work number to verify
   - [ ] Call emergency contacts to verify

2. **Medical Information**
   - [ ] Review all allergies with parent
   - [ ] Confirm medication names and dosages
   - [ ] Verify doctor contact information
   - [ ] Check medical aid card

3. **Documentation**
   - [ ] Request copy of medical aid card
   - [ ] Request immunization records
   - [ ] Request birth certificate copy
   - [ ] Request ID copy (parent)

4. **Consent Forms**
   - [ ] Media consent form
   - [ ] Medical treatment consent
   - [ ] Emergency transport consent
   - [ ] Field trip consent

---

## Post-Registration Tasks

After completing registration:

1. **File Management**
   - [ ] Create physical file for family
   - [ ] File all documentation
   - [ ] Create emergency contact card
   - [ ] Add to emergency contact list

2. **Staff Notification**
   - [ ] Inform all teachers
   - [ ] Notify kitchen staff of dietary restrictions
   - [ ] Alert staff to any special needs
   - [ ] Distribute emergency contact information

3. **Parent Communication**
   - [ ] Send welcome email
   - [ ] Provide parent handbook
   - [ ] Share school calendar
   - [ ] Explain communication procedures

4. **System Setup**
   - [ ] Verify data in system
   - [ ] Test parent login credentials
   - [ ] Ensure child appears in attendance
   - [ ] Check payment setup
